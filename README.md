# 📝 Notes App (React + Tailwind CSS)

A simple and responsive Notes App built using **React** and **Tailwind CSS**.
You can add notes, view them in a clean UI, and delete them easily.

---

## 🚀 Features

* ➕ Add Notes (Title + Details)
* 🗑️ Delete Notes
* 🎨 Beautiful UI with note cards
* 📱 Fully Responsive (Mobile + Desktop)
* ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

* React (useState)
* Tailwind CSS
* JavaScript (ES6)

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/notes-app.git
```

### 2. Navigate to project folder

```bash
cd notes-app
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the app

```bash
npm run dev
```

---

## 🎯 Usage

1. Enter title and details
2. Click **Add Note**
3. Notes will appear below
4. Click **Delete** to remove a note
